# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-03-25

### Added
- Initial release of Astro SaaS Landing Page template.

- Responsive Navbar with mobile menu.
- Dark mode support with theme toggle.
- Hero, Features, Pricing, and FAQ components.
- Interactive Testimonial Slider using Preact.
- Blog system with Content Collections support.
- SEO optimization (Meta tags, Sitemap, Robots.txt).
- Modern SaaS aesthetic with Tailwind CSS.
- TypeScript configuration and path aliases.
